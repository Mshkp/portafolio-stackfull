Clínica Virtual

Plataforma tecnológica para conectar pacientes con especialistas mediante consultas en línea y expedientes médicos digitales. [cite: 4, 5]

Stack del Proyecto

  • HTML / CSS / JavaScript
  • PHP / MySQL Workbench
  • XAMPP Server

Expedientes

Gestión digital de historias clínicas eliminando el uso de registros en papel.

Agendamiento

Sincronización de citas en tiempo real según disponibilidad.

Seguridad

Protección de historias clínicas con cifrado y autenticación.
